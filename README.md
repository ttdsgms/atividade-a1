# atividade a1

projeto da atividade de design adaptativo feito em astro, sem tailwind, com `pnpm`.

## o que tem aqui

- página principal com três versões da mesma interface:
  - desktop
  - tablet
  - mobile
- relatório em markdown

## rodar o projeto

```bash
pnpm install
pnpm dev
```

depois abre o endereço mostrado no terminal.

## build

```bash
pnpm build
pnpm preview
```

## estrutura

```text
src/
  components/
  layouts/
  pages/
  styles/
```

## páginas

- `/` -> layout principal da atividade
- `/relatorio` -> relatório em markdown

## observação

o pdf original da atividade continua na raiz da pasta como referência.
