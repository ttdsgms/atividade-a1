## 1. Objetivo

A proposta foi montar uma loja bem simples para mostrar a adaptacao de layout entre desktop, tablet e mobile. Eu mantive os mesmos elementos principais em todas as versoes para facilitar a comparacao.

## 2. Estrutura usada

Em todas as telas eu usei:

- logo
- campo de busca
- carrinho
- menu
- banner
- lista de produtos

Os produtos foram mantidos com o mesmo conteudo. O que muda de uma versao para outra e a organizacao do espaco.

## 3. Adaptacao por tela

- Desktop: quatro cards na mesma linha e menu completo.
- Tablet: duas colunas para os produtos e menu mais curto.
- Mobile: icones no topo, campo de busca separado e apenas uma coluna de produtos.

Preferi nao usar imagens de produto porque a atividade funciona bem so com cards de texto, o que deixou a pagina mais direta e mais facil de ajustar.

## 4. Conclusao

O resultado final ficou simples, mas suficiente para demonstrar a diferenca entre os tres formatos. A principal preocupacao foi manter leitura clara e repetir a mesma base visual em todas as versoes.
